# Client side

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

# Server side

## Project setup
```
npm install
```

### Start server with nodemon.
```
npm start
```
